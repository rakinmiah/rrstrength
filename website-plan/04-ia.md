# Phase 4 — Information Architecture

## Narrative arc (one line)
**Promise → Proof → Personality → Path → Ask** — a first-time visitor is hooked by a risk-free promise, immediately shown the coach is legitimate, connected to him as a person, walked through exactly what they'd get and how, then asked with the lowest-friction possible step.

## Signature module (declared)
**Module 6 — Results.** A `LAYERED` scroll-linked before/after reveal: as the visitor scrolls, a client's "before" image wipes to "after" via a clip-path tied to scroll progress, with the outcome stat and named client settling in. No competitor in the corpus shows visual results at all, let alone with motion craft — this is the page's memorable, un-transferable moment and the literal proof behind the contrarian thesis.

## Optional / variant modules (swap-ins, not additions)
- **Events module — no-event variant:** when no upcoming event exists, the same module renders a "Past events / community" recap (Ramadan Rumble) so it never shows a stale date.
- **Hero — seasonal variant:** the hero image can swap to a current-event promo crop without layout change.
